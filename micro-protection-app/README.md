# Micro-Protection Fluid — HSBC Life Prototype

Interactive mobile app & web dashboard prototype for the Micro-Protection Fluid product.

## Run locally

```bash
cd micro-protection-app
npm install
npm run dev
```

Open http://localhost:5173

- **/** — Choose mobile or web experience
- **/mobile** — Phone app (bottom nav, Alex's dashboard)
- **/web** — Desktop dashboard (sidebar layout)

## Features

### Mobile & Web (shared)
- **Home** — Live donut chart, AI insight card, scenario switcher (Tokyo, WFH, Keeta, Guangzhou HSR, Typhoon, Climbing)
- **Activity** — Reallocation history timeline
- **My Protection** — Category breakdown, floors, payout caps
- **Insights** — AI notification feed
- **Profile** — Plan info & stats

### Sub-pages (from quick actions)
- **Claims** — Claim history + file claim
- **Family Protection** — Linked family members
- **Settings** — Personal floors, notifications, privacy
- **Coverage details** — Full allocation + multipliers

## Demo tip

Use the **scenario picker** on Home to watch the donut chart and insight card update in real time.
