# Micro-Protection Fluid — HSBC Life

Interactive **mobile app** and **web dashboard** prototype for HSBC Life's Micro-Protection Fluid product.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

- **/** — Choose mobile or web
- **/mobile** — Phone app
- **/web** — Desktop dashboard

## Stack

React · Vite · Tailwind CSS · Recharts · React Router
